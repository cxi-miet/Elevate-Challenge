Round 1

You have to fullfill these minimum requirements.

  • You have to connect two computers with each other.
  • Assign the ip address to both the computers.
  • There should be connectivity between two computers.

Round 2

You have to fullfill these minimum requirements.

  • You have to connect 5 computers with each other using switch.
  • You have to provide the ip addresses to the computers.
  • There should be connectivity between two computers.

Round 3

You have to fullfill these minimum requirements.

  • You have to connect 15 computers with each other using switch.
  • You have to configure the DHCP server to assign the ip addresses to the computers.
  • There should be connectivity between two computers.

Round 4

You have to fullfill these minimum requirements.

  • You have to make two networks consist of 4 computers.
  • You have to connect two different network using router.
  • You have to configure the DHCP server for one of the network.
  • You have to provide the ip addresses to all the computers.
  • You have to use same ip block as shown in the image.
    1. 192.168.2.0
    2. 192.168.1.0
  • There should be connectivity between two computers.
